# 1. A simple API with Node.js and Fastify

## 1.1 Requirements

- Node.js 12+
- npm 6+
- Docker 19+ (optional)

## 1.2 Setup

### 1.2.1 Initialize the project

```bash
npm init -y
```

### 1.2.2 Install dependencies

```bash
npm i fastify
```

```bash
npm install postgres
```

```bash
npm install dotenv -D
```
